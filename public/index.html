Welcome to the KBTG AI Backend Workshop! This is a simple REST API built with Go and Fiber.

Available Endpoints

GET /api/v1/health

Health check endpoint

GET /api/v1/hello?name=YourName

Simple hello world endpoint with optional name parameter

GET /api/v1/users

Get all users

GET /api/v1/users/:id

Get a specific user by ID

POST /api/v1/users

Create a new user (send JSON: {"name": "Name", "email": "email@example.com"})

PUT /api/v1/users/:id

Update a user (send JSON: {"name": "New Name", "email": "newemail@example.com"})

DELETE /api/v1/users/:id

Delete a user by ID
